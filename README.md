# Codex Harness v0 — Delivery Bot Zero

이 저장소는 작은 Unity 2D 퍼즐 게임 Delivery Bot Zero를 사용해 최소 Codex 하네스 기반 개발 워크플로우를 검증하는 실험입니다.

## 목표

Codex를 사용해 Unity 게임 개발을 문서화, 계획, 구현, 검증하는 흐름을 검증합니다.

## 현재 단계

M0: Unity 프로젝트 생성, GitHub 업로드, 최소 Codex 하네스 구성.

## 다음 단계

M0 검증이 완료되면 다음 단계는 M1: First Playable Loop입니다.

M1은 별도 ExecPlan에서 `Start -> Move -> Pickup -> Deliver -> Result -> Retry` 루프를 구현합니다. 이 README는 M1이 이미 시작되었거나 구현되었음을 의미하지 않습니다.

## 아직 포함하지 않는 것

- Gameplay implementation
- Unity MCP
- custom skills
- hooks
- subagents
- external packages
