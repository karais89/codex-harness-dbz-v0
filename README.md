# Codex Harness v0 — Delivery Bot Zero

이 저장소는 작은 Unity 2D 퍼즐 게임 Delivery Bot Zero를 사용해 최소 Codex 하네스 기반 개발 워크플로우를 검증하는 실험입니다.

## 목표

Codex를 사용해 Unity 게임 개발을 문서화, 계획, 구현, 검증하는 흐름을 검증합니다.

## 현재 단계

M3 완료: M1/M2 플레이어블 루프를 반복 검증 가능한 상태로 정착했다.

## 다음 단계

다음 단계는 M4: Handoff Test입니다.

M4는 새 Codex 세션이 repo 문서만 읽고 프로젝트 목적, 완료 상태, 현재 마일스톤, 다음 작업, 남은 위험을 정확히 이어받을 수 있는지 검증합니다.

## 아직 포함하지 않는 것

- Unity MCP
- custom skills
- hooks
- subagents
- external packages
