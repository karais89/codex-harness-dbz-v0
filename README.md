# Codex Harness v0 — Delivery Bot Zero

이 저장소는 작은 Unity 2D 퍼즐 게임 Delivery Bot Zero를 사용해 최소 Codex 하네스 기반 개발 워크플로우를 검증하는 실험입니다.

## 목표

Codex를 사용해 Unity 게임 개발을 문서화, 계획, 구현, 검증하는 흐름을 검증합니다.

## 현재 단계

M4 완료: 새 Codex 세션이 저장소 문서만 읽고 현재 상태를 이어받을 수 있음을 확인했다.

## 다음 단계

다음 마일스톤은 M5: 공개 가능한 포트폴리오 패키징입니다.

M5는 지금까지 검증한 Codex 하네스 흐름과 현재 Unity Play Mode 루프를 공개 가능한 형태로 정리하는 단계입니다.

## 아직 포함하지 않는 것

- Unity MCP
- custom skills
- hooks
- subagents
- external packages
