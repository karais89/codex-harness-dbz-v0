# Codex Harness v0 — Delivery Bot Zero

이 저장소는 작은 Unity 2D 퍼즐 게임 Delivery Bot Zero를 사용해 최소 Codex 하네스 기반 개발 워크플로우를 검증하는 실험입니다.

## 목표

Codex를 사용해 Unity 게임 개발을 문서화, 계획, 구현, 검증하는 흐름을 검증합니다.

## 현재 단계

M2 완료: M1 루프 유지, UI 읽기성 개선, Unity Play Mode 수동 회귀 검증 완료.

## 다음 단계

다음 단계는 M3: 검증/테스트 루프 정착입니다.

M3는 별도 논의와 ExecPlan에서 반복 가능한 Play Mode 체크리스트와 가능한 경우 간단한 Edit Mode Test를 검토합니다.

## 아직 포함하지 않는 것

- Unity MCP
- custom skills
- hooks
- subagents
- external packages
