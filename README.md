# Codex Harness v0 — Delivery Bot Zero

이 저장소는 작은 Unity 2D 퍼즐 게임 Delivery Bot Zero를 사용해 최소 Codex 하네스 기반 개발 워크플로우를 검증하는 실험입니다.

## 목표

Codex를 사용해 Unity 게임 개발을 문서화, 계획, 구현, 검증하는 흐름을 검증합니다.

## 현재 단계

M1 완료: First Playable Loop 구현과 Unity Play Mode 수동 검증 완료.

## 다음 단계

다음 단계는 M2: Change Request입니다.

M2는 별도 논의와 ExecPlan에서 기존 M1 기능을 깨지 않고 작은 변경을 수행할 수 있는지 검증합니다.

## 아직 포함하지 않는 것

- Unity MCP
- custom skills
- hooks
- subagents
- external packages
